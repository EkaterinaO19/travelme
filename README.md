## Run Locally

Clone the project

```bash
  git clone https://github.com/EkaterinaO19/travelme.git
```

Go to the project directory

```bash
  cd travelme
```

Install dependencies

```bash
  npm init
```

Start the server

```bash
  npm run start
```
